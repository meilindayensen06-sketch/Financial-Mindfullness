Kuesioner Financial Mindfulness

Petunjuk: Pilih satu jawaban untuk setiap pernyataan.
Skala: 1 = Sangat Tidak Setuju, 2 = Tidak Setuju, 3 = Netral, 4 = Setuju, 5 = Sangat Setuju.

1. Saya memperhatikan setiap kali saya mengeluarkan uang.

2. Saya tahu kemana sebagian besar uang saya dibelanjakan.

3. Saya mampu menahan diri dari belanja yang tidak penting.

4. Saya membuat catatan atau anggaran keuangan secara rutin.

5. Saya merasa tenang dan sadar saat mengambil keputusan keuangan.

6. Saya lebih mengutamakan kebutuhan dibandingkan keinginan.

7. Saya memikirkan konsekuensi jangka panjang sebelum membeli sesuatu.

8. Saya merasa bersyukur dengan kondisi keuangan saya saat ini.

Semua pertanyaan wajib diisi.
